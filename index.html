My Projects

Pill Identification and Detection

This project aims to enhance patient safety by minimizing errors from incorrect prescriptions or tampered pill labels through pill classification using machine learning. It employs Convolutional Neural Networks (MobileNetV2) via TensorFlow and Keras to analyze pill characteristics like size, color, and shape from uploaded images. The system identifies the pill, providing its name, uses, and dosage recommendations.

Audio to Text

This project focuses on converting audio inputs into accurate text using advanced speech recognition technologies. By leveraging machine learning algorithms and natural language processing, it aims to transcribe spoken words into text efficiently. The system supports various use cases, including transcription services, accessibility tools, and real-time communication enhancement.

Weather or Not

Weather or Not is a project that provides simple, accurate weather updates and forecasts. It helps users plan their daily activities by offering real-time weather information in an easy-to-understand format. The project focuses on accessibility and convenience for everyday use.
